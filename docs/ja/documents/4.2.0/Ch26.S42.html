26.42 トップレベル環境

ライブラリで定義されているよく使う関数はトップレベルで再定義されている． 本説では，Standard ML標準ライブラリで定義されている型や関数の中でトップレベルで 再定義されているものを示す．

    infix宣言

      infix 7 * / div mod
  infix 6 + - ^
  infixr 5 :: @
  infix 4 = <> > >= < <=
  infix 3 := o
  infix 0 before
